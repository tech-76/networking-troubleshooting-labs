# Networking Troubleshooting Labs

This repository includes networking notes and troubleshooting examples for IT support and junior network administration roles.

## Topics Covered

- TCP/IP fundamentals
- DNS troubleshooting
- DHCP troubleshooting
- Subnetting basics
- Ping, traceroute, ipconfig, and nslookup
- VPN troubleshooting
- Cisco Packet Tracer lab notes

## Skills Demonstrated

- Network troubleshooting
- Diagnostic commands
- Technical documentation
- Network support fundamentals
